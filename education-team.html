Setup Supabase diperlukan

Dashboard ini butuh database Supabase supaya tugas & komentar tersimpan bersama untuk semua anggota tim. Langkah:

  1. Buat project gratis di supabase.com.
  2. Buka SQL Editor, jalankan isi file supabase/schema.sql dari repo ini.
  3. Buka Project Settings → API, salin Project URL dan anon public key.
  4. Isi kedua nilai itu ke file supabase-config.js di root repo.
  5. Commit & push — Vercel akan otomatis deploy ulang.
