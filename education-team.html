Setup diperlukan

Dashboard ini membaca tugas dari Google Spreadsheet dan menyimpan komentar & status ke GitHub. Langkah:

  1. Buka Google Spreadsheet data tugas → klik Share → ubah ke "Anyone with the link" → Viewer.
  2. Buka github.com → Settings → Developer settings → Personal access tokens.
  3. Klik Generate new token (classic). Beri scope repo. Salin token.
  4. Buka Vercel dashboard → project ini → Settings → Environment Variables.
  5. Tambah variable: GITHUB_TOKEN = token yang tadi disalin.
  6. Klik Redeploy dari tab Deployments supaya environment variable terbaca.
