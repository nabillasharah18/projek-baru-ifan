Setup GitHub Token diperlukan

Dashboard ini menyimpan data tugas & komentar ke file JSON di repo GitHub lewat Vercel serverless function. Langkah:

  1. Buka github.com → Settings → Developer settings → Personal access tokens → Fine-grained tokens.
  2. Klik Generate new token. Pilih repo projek-baru-ifan, beri permission Contents: Read and write.
  3. Salin token yang dihasilkan.
  4. Buka Vercel dashboard → project ini → Settings → Environment Variables.
  5. Tambah variable: GITHUB_TOKEN = token yang tadi disalin.
  6. Klik Redeploy dari tab Deployments supaya environment variable terbaca.
